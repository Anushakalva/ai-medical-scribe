# 🩺 AI Medical Scribe

An AI-powered Medical Scribe application that converts doctor-patient conversations into structured clinical documentation, reducing manual note-taking and improving workflow efficiency.

> 🚧 This project is currently under active development. New features and improvements are being added regularly.

---

## 📌 Overview

AI Medical Scribe is a full-stack healthcare application designed to assist doctors by automatically generating structured medical notes from patient consultations.

The application records conversations, processes them using AI-powered speech recognition and natural language processing, and generates organized SOAP notes while highlighting important clinical information.

---

## ✨ Current Features

- 🎤 Audio recording for consultations

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Anushakalva/ai-medical-scribe.git
```

### Navigate to the Project

```bash
cd ai-medical-scribe
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd ../backend
npm install
```

### Install AI Service Dependencies

```bash
cd ../ai-service
pip install -r requirements.txt
```

### Start the Application

Frontend

```bash
npm run dev
```

Backend

```bash
npm start
```

AI Service

```bash
python app.py
```
